Para poder entrar, você precisa de uma chave grátis da Groq. É rapidinho, veja como:

  2. Crie uma conta grátis (com email ou Google)
  3. Clique em "Create API Key"
  4. Dê um nome qualquer pra ela e confirme
  5. Copie a chave gerada (começa com gsk_)
  6. Cole ela no campo abaixo e clique em confirmar
